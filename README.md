# Haridev P - Portfolio

This repository contains the source code for my personal portfolio website, built with React, Tailwind CSS, and Vite.

## Project Structure

The project follows a structured architecture:

- **`src/components`**: reusable React components (App, WindowFrame, Terminal, etc.)
- **`src/styles`**: Global styles and Tailwind configuration.
- **`src/scripts`**: Application entry points and utility scripts (e.g., API calls).
- **`src/data`**: Static data for the portfolio (resume, blog posts, config).
- **`public`**: Static assets served at the root.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Features

- **Cybersecurity Theme**: A terminal-like interface with "hacker" aesthetics.
- **AI Integration**: "Tactical AI" assistant powered by Google Gemini.
- **Interactive Dashboard**: Glitch effects, typing animations, and window management.
- **Responsive Design**: Optimized for both desktop and mobile views.

## Deployment

This project is configured for deployment on GitHub Pages.

```bash
npm run deploy
```
